# Aditanar Institution ERP Portal

## Overview
This is a comprehensive college ERP (Enterprise Resource Planning) portal designed for Aditanar Educational Institution. The portal provides a unified platform for students, applicants, parents, administrators, and teachers to access various academic and administrative services.

## Features
- **Multi-role Authentication**: Separate login portals for different user types
- **Student Dashboard**: Access to profile, courses, grades, and attendance
- **Applicant Portal**: Application submission and tracking
- **Parent Portal**: Child progress monitoring and communication
- **Admin Panel**: Administrative controls and management
- **Teacher Portal**: Course management and student evaluation

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Responsive Design

## Installation
1. Clone or download the project files
2. Open `index.html` in a web browser
3. No additional setup required - pure frontend implementation

## Usage
- Navigate to the home page
- Select your role (Applicant, Student, Parent/Guardian, Administrator, Teacher/Lecturer)
- Login with appropriate credentials
- Access role-specific features and information
